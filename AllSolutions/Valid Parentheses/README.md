# Condition And Task
Write a function that takes a string of parentheses, and determines if the order of the parentheses is valid. The function should return **true** if the string is valid, and **false** if it's invalid.

## Examples
> "()"              =>  true<br>

> ")(()))"          =>  false<br>

> "("               =>  false<br>

> "(())((()())())"  =>  true
